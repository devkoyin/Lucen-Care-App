Shadow tokens — new

4-level elevation scale. This is what fixes "flat". Cards get shadow-sm, modals get shadow-lg.

shadow-xs
0 1px 2px rgba(0,0,0,.05)
badges, chips, subtle borders
shadow-sm new
0 2px 4px rgba(0,0,0,.06), 0 1px 2px rgba(0,0,0,.04)
cards, inputs on focus
shadow-md new
0 4px 8px rgba(0,0,0,.07), 0 2px 4px rgba(0,0,0,.04)
popovers, dropdowns, hero card
shadow-lg new
0 10px 20px rgba(0,0,0,.08), 0 4px 8px rgba(0,0,0,.04)
modals, bottom sheets, floating elements
Also adding --card-shadow: var(--shadow-sm) so all cards get lift by default — no more flat bordered rectangles.
